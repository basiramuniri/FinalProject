CIS 130 – My Web Development Resources

A collection of everything that helped me learn web development this semester.

Welcome

I took CIS 130 because I wanted to understand how websites actually work — not just use them, but build them from scratch. Going into this course I had no experience with HTML or CSS at all, so it was a real learning curve. Looking back though, I am glad I stuck with it because I learned a lot more than I expected.

This website is a collection of the resources, videos, and tools that helped me the most throughout the semester. Each page covers a different topic from the course. I hope it can be useful to anyone else learning web development too.
